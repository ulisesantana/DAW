Ejercicio 3

Desarrolla una simple aplicación web en PHP llamada cookiebuscador.php que permita realizar búsquedas a través de Google. La aplicación recogerá de un formulario la búsqueda a realizar, y redirigirá la página hacia el buscador de Google empleando la siguiente URL para realizar la búsqueda en Internet: 

http://www.google.com/search?q=busqueda

La aplicación deberá además almacenar en una cookie la última búsqueda realizada por el usuario para que se le presente en el formulario cuando éste vuelva a acceder la web.
